# AskBox

AskBox is a modern, lightweight, and powerful form-building platform inspired by Google Forms. It allows users to easily create **quizzes, surveys, polls, feedback forms, and data-collection workflows** with a clean and intuitive interface.

Built with the MERN stack, AskBox focuses on speed, flexibility, and great user experience. Users can build custom forms, share them publicly or privately, and analyze responses through a simple, organized dashboard.
